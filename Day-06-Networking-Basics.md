# Day 6 – Networking Basics

## Commands Practiced

- hostname
- hostname -I
- ip a
- ping
- curl -I
- wget
- cat

## Key Learnings

- hostname → Shows system name
- hostname -I → Shows IP address
- ip a → Displays network interfaces
- ping → Tests network connectivity
- curl → Retrieves data from websites
- wget → Downloads files from the internet
