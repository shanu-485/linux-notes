# Day 5 – Process Management

## Commands Practiced

- ps
- ps -ef
- top
- sleep
- jobs
- fg
- kill

## Key Learnings

- A process is a running program.
- Every process has a PID.
- `top` monitors processes in real time.
- `jobs` shows background jobs.
- `kill` stops a process using its PID.
